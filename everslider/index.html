Everslider as "Image Slider"

When number of visible slides is limited to one, Everslider carousel turns into responsive slider that can be used as image slider. And in "normal" mode plugin will stop of first/last slide.

// Image slider
$('#image_slider').everslider({
	mode: 'normal',
	maxVisible: 1
});

Everslider as "Featured Carousel"

Everslider in "carousel" mode can help you easily organize your featured works or recent blog posts into infinite carousel. Change your slides size instantly using plugin options - itemWidth, itemHeight and itemMargin. Use mousewheel or touchswipe gesture to scroll carousel!

Everslider as "Fullwidth Carousel"

By setting slide height to 'auto' in css and itemKeepRatio plugin option to 'false' you will have slides of any height determined by the content itself. Carousel in this example works in "circular" mode and enables keyboard arrows navigation, so you can test it here.
